# GoZon

GoZon — это микросервисная система, демонстрирующая надёжное взаимодействие между сервисами с использованием паттернов Outbox и Inbox через Apache Kafka.

# Спецификация
https://elegant784seat.github.io/gozon/

## Архитектура

Система состоит из следующих компонентов:

- **order-service** — управляет заказами и публикует события о создании заказа.
- **payments-service** — обрабатывает платежи, подписываясь на события заказов.
- **gateway** — API-шлюз для внешнего доступа к сервисам.
- **PostgreSQL** — отдельные базы данных для каждого сервиса.
- **Apache Kafka** — брокер сообщений для асинхронной передачи событий.

Все сервисы используют изолированные базы данных и общаются только через Kafka, что обеспечивает слабую связанность и устойчивость к сбоям.
