# Архитектура системы UserFlow

## Обзор
Система управления студенческими работами с проверкой на плагиат.

## Микросервисы

### 1. Gateway Service (Порт: 8080)
- Единая точка входа для клиентов
- Маршрутизация запросов
- Агрегация ответов
- Документация API (Swagger UI)

### 2. Storage Service (Порт: 9090)
- Управление файлами работ
- Хранение в S3/MinIO
- Метаданные файлов в PostgreSQL

### 3. Analysis Service (Порт: 9091)
- Анализ работ на плагиат
- Хэширование SHA-256
- Сохранение отчетов

## Технологии
- **Бэкенд:** Spring Boot 3, Java 21
- **База данных:** PostgreSQL
- **Хранилище:** MinIO (S3-совместимое)
- **Документация:** SpringDoc OpenAPI 3
- **Контейнеризация:** Docker, Docker Compose
